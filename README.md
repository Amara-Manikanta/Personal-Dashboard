# Personal Dashboard

A comprehensive personal dashboard application built with React to track novels, manage writing projects, and explore travel destinations in India.

## Features

### 📚 Novels Dashboard
- **Library Tracking**: Keep track of novels you've read, are reading, or want to read.
- **Detailed Views**: View detailed information about each novel, including descriptions, quotes, and ratings.
- **Search & Filter**: Easily find books by title, author, or genre.

### ✍️ Writing Dashboard
- **Story Management**: Organize and write your own stories.
- **Part Tracking**: Manage multiple parts of a story with original and modified versions.

### 🇮🇳 Travel Tracker
- **Indian States**: Explore and track visits to different states in India.
- **Interactive Map**: Visual representation of tracked locations.

## Tech Stack
- **Frontend**: React (Vite), JavaScript (ES6+)
- **Styling**: Vanilla CSS
- **Data Storage**: Local JavaScript data files
